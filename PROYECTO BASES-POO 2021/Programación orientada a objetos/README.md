## _Instalación_
**Versión de framework compatible:** .NET 5

**Es necesario tener una [base de datos](https://github.com/UCASV/proyecto-final-grupo34/blob/master/PROYECTO%20BASES-POO%202021/Bases%20de%20datos/Banco%20de%20datos.sql) existente:** Para el correcto funcionamiento del programa, lo mejor es tener a disposición una base de datos previamente creada.

**Gestor de bases de datos utilizado:** Microsoft SQL Server

**Pasos para instalar el programa:**

1. En la carpeta donde se encuentra el instalador llamada ProyectoFinal_Installer, daremos doble click para abrirlo.

![][Imagen1]

2. Se abre una ventana y le damos "Next".

![][Imagen2]

3. Seleccionamos nuevamente "Next".

![][Imagen3]

4. Nos confirma que el programa está listo para ser instalado y presionamos "Next".

![][Imagen4]

5. Aceptamos los permisos necesarios y esperamos a que termine de instalarse.

![][Imagen5]

6. El programa se instaló exitosamente, le damos a "Close".

![][Imagen6]

7. Nos dirigimos a nuestra escritorio y vemos que se creó un acceso directo del programa, le damos click.

![][Imagen7]

8. Listo, el programa fue instalado y abierto correctamente.

![][Imagen8]
